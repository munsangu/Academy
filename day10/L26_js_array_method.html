Javascript Array Method

const a = ["b", "c", "a", "z", "f", "e", "h"]

  • a.toString( ) : 쉼표로 구분된 배열의 값을 문자열로 출력
  • a.join("-") : 배열의 값을 ( )안의 문자열로 결합한 값으로 출력
  • a.pop( ) : 배열의 마지막 값을 반환하고 제거
  • a.push( ) : 배열의 마지막에 새로운 값을 추가
  • a[a.length] = value : 배열의 마지막에 새로운 값을 추가
  • array1.concat() : 배열의 끝에 새로운 값(배열)을 추가 및 반환
  • a.shift( ) : 배열의 첫번째 값을 반환하고 제거 (모든 index를 한 칸 당김)
  • a.unshift( ) : 배열의 첫번째 값을 새 값을 추가 (모든 index를 한 칸 밀림)
  • a.slice( ) : 지정한 시작점부터 끝점 까지의 값을 반환 (원래 배열 유지)
  • a.splice( ) : 기존의 값을 삭제 또는 교체하거나 새 값을 추가 (원래 배열 손상)


  • a.sort( ) : (정렬기준 = 유니코드) 배열의 값을 적절한 위치에 정렬한 후 반환 (원래 배열 손상)
  • a.reverse( ) : (정렬기준 = 유니코드) 배열의 순서를 반전 (원래 배열 손상)
