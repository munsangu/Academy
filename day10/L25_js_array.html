Javascript Array

여러 변수를 index로 참조해 저장하는 데이터 타입 (Iterator 상속)

작성법 : const(let) array = [item1, item2, item3, ...]

호출법 : array[index]

["A", "B", "C", "D"].toString() : (객체를 바로 호출하면 toString()이 생략됨)

대입법 : array[index] = value ex) a[1] = "b" :

typeof [] : (배열의 타입은 typeof 사용 불가)

Array.isArray([]) :

Array[max index + 1]= value : (Javascript는 동적으로 배열의 길이를 조절)
