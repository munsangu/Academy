JSON(JavaScript Object Notation)

  • Javascript의 객체 표현법
  • JSON은 문자열로 표현될 수 있음
  • 아주 가벼운 경량 데이터 교환 방식으로 문자열로만 이루어짐 (모든 프로그래밍 언어에 독립적)
  • 데이터를 주고 받기 위해 사용됨

JSON 작성법

  • JSON : "{ }"
  • const jsonData = "{ }";
  • const jsonData = '{"key1" : value1, "key2" : value2, ... }'; -> key의 이름 중복 불가
  • json 형식의 문자열을 Javascript에서 JSON.parse로 Object type으로 변경

Javascript - JSON

  • Javascript에서 JSON Object( )의 프로퍼티 설명서
  • new Object( )로 JSON 생성 가능
  • Javascript에서 JSON을 기본 데이터 타입처럼 iteral한 방식을 제공
  • {key : value} : literal 방식(Javascript 문법에서 사용되는 JSON)
  • "{'key': 'value'}" : 문자열로 표현된 JSON (서버에서 주고받는 데이터)
  • Javascript는 함수 지향 언어로 함수를 객체로 선언할 수 있음
  • function Person(name, age) {this.name = name, this.age = age} => new Person("TonyStark", "55")
  • Javascript 객처의 프로퍼티 설명들이 문자열 데이터로 전송되는 것 = JSON
