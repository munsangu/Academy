html의 inline 요소

inline 요소느누 block에 포함된 요소로 넓이와 높이가 없으며 텍스트(contents) 영역에서 사용됨

a, abbr, acronym, b, bdo, big, br, button, cite, code, dfn, em, i, img, input, kbd, label, map, object, q, samp, script, select, span, strong, sub, sup, textarea, tt, var

html 문자의 스타일을 지정하는 inline 요소

b - 굵은 텍스트 font-weight: bold;
strong - 중요한 텍스트 font-weight: bold;
i - 기울인 텍스트 font-style: italic;
em - 기울인 텍스트 + 강조 font-style: italic;
mark - 표시된 텍스트 background-color: yellwo; color: black;
small - 작은 텍스트 font-size: smaller;
del - 삭제될 텍스트 text-decoration: line-through;
ins - 삽입할 텍스트 text-decoration: underline;
sup - 윗 첨자 텍스트 vertical-align: sup; font-size:smaller; ex) 32=9;
sub - 아래 첨자 텍스트 vertical-align: sub; font-size:smaller; ex) h2o

인용을 나타내는 inline 요소

abbr -  HTML  약어 또는 두문자어를 정의
bdo -   fedcba 텍스트 방향을 정의
cite -   작품의 제목 을 정의
q   -   큰 따옴표 인용을 정의
