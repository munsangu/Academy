class 상속으로 재사용을 배우자

  • 상속은 부모의 타입(멤버)을 물려받는 것
  • 상속 받았을 때 부모가 생성자를 작성한 경우 꼭!! 부모의 생성자를 호출해야한다.(super())
  • 멤버의 이름은 중복될 수 없다.
  • get 멤버() 선언하면 멤버가 속성이 추가된다.(멤버로 한번 더 선언하면 안된다.)
  • 예)class A{ name; get name(){};}(X) class{ get name(){}}(O)
  • 예)class A{ #name; get name(){};}(o) 자바스크립트에서 getter setter를 선언한 멤버는 private 한다.
  • 부모의 함수를 재정의 하려면 함수에서 부모의 함수를 다시 호출해야한다.
