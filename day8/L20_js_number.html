Number의 연산과 자동 형 변환

숫자는 연산을 하기위해 존재하고 Javascript의 숫자는 숫자가 아닌 것도 연산하기 위해 형변환을 시도 ex)"10" -> 10

+ 연산은 문자열을 합치는 연산이기도 함, 숫자의 + 연산 시 문자열이 있다면 숫자를 문자열로 형변환

  • 10 + 20 :
  • 10 + "20" :
  • "10" + "20" :
  • "A" + 10 + 20 :
  • 10 + 20 + "A" :
  • 1 + true :
  • 1 + false :
  • 1 + null :
  • 1 + undefined :
  • 1 + NaN :

+ 연산을 제외한 나머지 연산은 문자열이 나올 시 자동 형 변환을 시도함
만약에 형변환되지 않으면, 숫자가 아니라는 뜻의 NaN를 출력함
false와 null은 0으로 변환되고, NaN과 undefined는 NaN이 됨
