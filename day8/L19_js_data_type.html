Javascript의 데이터 타입과 기본 데이터 타입

Javascript의 변수는 모든 데이터 타입을 참조할 수 있음.(동적 타입, Dynamical type) 이 때 모든 프로그래밍 언어와 같이 자주 쓰이는 타입을 기본 데이터 타입으로 지정하고 있으며 총 7가지(string, number, bigint, boolean, symbol, null, undefined)가 있음

null은 '타입'이 정해지지 않은 상태

기본 데이터 타입은 아니지만 기본 데이터 타입처럼 사용할 수 있는 배열과 JSON

let arr = [a, b, c, d];(new 없이 생성 가능한 배열타입) arr = new Array(a, b, c, d)

배열은 순서를 가지는 데이터 타입으로 모든 데이터 타입을 참조할 수 있음(같은 데이터 타입을 참조하는 것을 권장)

let json = {key:value, key2:value2, ...} : new 없이 생성가능한 JSON 타입(new Object())

json은 key와 value를 가지는 데이터 타입으로 new Object로 생성 가능 이때 Object는 모든 Class의 부모인 Object와 별개의 Class

Javascript Object Notation의 줄임말로 경량 데이터 교환 형식(언어에 독립적으로 문자열 형식이 중요)
