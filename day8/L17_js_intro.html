Javascript Introduce

Javascript는 Web Browser 상에서 UI를 동적으로 보여주기 위해 개발되었으나 배우기 쉽고 개발 속도가 빨라서 서버(Node.js)나 모바일앱(React-Native, NativeScript) 등 프로그래밍의 여러 분야에서 사용되고 있음 또한, 매해 새 버전을 출시해 오류를 수정하거나 객체지향적 특성을 발전시키고 있음

브라우저 언어인 Javascirpt의 특징

  • Javascript는 객체 기반의 *스크립트 언어*로 브라우저에서 사용됨
    (*스크립트 언어*: 응용프로그램을 동작시키는 언어의 통칭)
  • Javascript는 인터프리터 언어로 *컴파일*이 되지 않고 바로 해석됨
    (*컴파일*: 작성한 코드를 기계어로 해석하는 것, 오류가 있는 코드는 컴파일 되지 않음)
    이로 인해 동적으로 개발할 수 있지만, 실행 속도가 느리고 오류가 있는 페이지를 배포할 가능성이 있음
  • Javascript는 함수형 프로그래밍 언어였으나, ES6이후, Class와 변수를 추가해 객체 지향 언어의 특징을 가짐
  • Javascript는 주로 다루는 객체가 window.document로 DOM Tree로 구성된 document의 특정 Element를 찾는 것에서부터 시작됨 ex) documnet.getElementById()
  • Javascript는 HTML 문서가 모든 브라우저에 출력되고 마지막에 실행됨

Javascript가 브라우저에서 할 수 있는 일

  • HTML 문서의 내용과 속성을 추가 * 수정 * 삭제가 가능
  • HTML 문서의 스타일을 변경할 수 있음
  • HTML 요소에서 발생하는 외부 기기의 입력을 이벤트로 받아 UI를 동적으로 바꿈
  • 브라우저의 (BOM; Browser Of Model) 정보를 알 수 있고, 입력 및 수정을 할 수 있음

HTML에서 Javascript 작성법

  • script 태그 내부 or script 파일에 작성
  • script type="text/JavaScript" -> old javascript로 더 이상 사용되지 않음
  • script scr="" 확장자가 js인 파일을 외부에서 참조 가능
  • 코드를 실행 및 선언 후 꼭 세미콜론 작성
  • //: 한줄 주석 & /* */: 여러줄 주석
  • "use strict": 엄격모드(es6 사용 시 자동으로 설정됨 ex>Class, let, const ...)
    js 파일 상단에 꼭 작성해야 함(과거 문법이 오류를 많이 만들기 때문)
    var num = 10; (O) num = 20(△)
