Javascript String.method()

문자열은 유저에게 전달하는 데이터로 웹 상에서 가장 많이 사용됨 따라서 문자열을 다루는 함수가 많이 존재

method 설명 예시
toUpperCase() / toLowerCase() 대문자 / 소문자로 변환 "ironman".toUppecase() :
"IRONMAN".toLowerCase() :
trim() 앞뒤 공백 제거 " IronMan ".trim() :
charAt() / charCodeAt() 문자열에서 특정 인덱스에 위치하는 유니코드 단일문자를 반환 / 주어진 인덱스에 대한 UTF-16 코드를 출력 "ABCDE".charCodeAt(0) :
"ABCDE".charAt(0) :
"ABCDE"[0] :
slice() 문자열의 일부를 추출하면서 새로운 문자열을 반환 "I'm IronMan".slice(0,3) :
"I'm IronMan".slice(4) :
"I'm IronMan".slice(-7) :
substring() slice()와 동일한 기능이나 양수 index만 사용 가능 "I'm IronMan".substring(4) :
substr() 일부 문자 추출, 시작점부터 길이까지 추출 "I'm IronMan".substr(0,3) : "I'm IronMan".substr(-12,4) :
startsWith() 특정 문자열로 시작하는지 여부 (결과값: Boolean) "I'm IronMan".startsWith("I") :
"I'm IronMan".startsWith("I'm") :
endsWith() 끝에서부터 특정 문자열로 끝나는지 여부 (결과값: Boolean) "I'm IronMan".endsWith("n") :
"I'm IronMan".endsWith("Iron") :
includes() (위치 지정 가능) 특정 문자열의 포함 여부 검색 (결과값: Boolean) "I'm IronMan".includes("Iron") :
"I'm IronMan".includes("Thor") :
indexOf() 주어진 값과 일치하는 첫번째 index를 반환(일치하는 값이 없으면 -1) "I'm IronMan".indexOf("Iron") :
"I'm IronMan".indexOf("Thor") :
lastIndexOf() 주어진 값과 일치하는 부분을 역순으로 탐색해 최초로 마주치는 index를 반환(일치하는 값이 없으면 -1) "I'm IronMan".lastIndexOf("I") :
"I'm IronMan".lastIndexOf("I",0) :
split() 지정한 구분자를 이용하여 여러 개의 문자열로 나누어 배열로 만듦 let marvel = "Ironman, Hurk, Thor, Captain, Deadpool"
marvel.split(",") :
marvel.split(",", 2) :
replace() 어떤 패턴에 일치하는 일부 또는 모든 부분이 교체된 새로운 문자열을 반환
(기존 데이터에는 변화가 없음) x !== x.replace() 		let marvel = "Ironman, Hurk, Thor, Captain, Deadpool"
marvel.replace("Deadpool", "Hwak-eye") :
replaceAll() 어떤 패턴에 일치하는 값을 모든 부분이 교체된 새로운 문자열을 반환
(기존 데이터에는 변화가 없음) x !== x.replaceAll() 		let marvel = "Ironman, Hurk, Thor, Captain, Deadpool"
marvel.replaceAll("Deadpool", "Hwak-eye") :
