할당 연산자와 증가 및 감소 연산자

operator(연산) 이름 설명 예시
+= 추가 할당 숫자 = 숫자 + 추가값 -> 자기 자신에 추가값을 더함(더할 값을 대입) x += 10 (x = x + 10)
-= 빼기 할당 숫자 = 숫자 - 뺄값 -> 자기 자신에 뺄값을 뺌(뺄 값을 대입) x -= 10 (x = x - 10)
*= 곱셈 할당 숫자 = 숫자 * 곱할값 -> 자기 자신에 곱할값을 곱함(곱할 값을 대입) x *= 10 ( x= x * 10)
/= 분할 할당 숫자 = 숫자 / 나눌값 -> 자기 자신에 나눌값을 나눔(나눌 값을 대입) x /= 10 ( x= x / 10)
++a / --a 후위 (증가 / 감소) 연산자 a = a + 1 / a = a - 1 -> 숫자 앞에 증가 및 감소 연산자가 오면 연산을 먼저 한 후, 다른 것을 실행 a(++i) -> i를 1 증가시키고 a()를 실행
a(--i) -> i를 1 감소시키고 a()를 실행
a++ / a-- 전위 (증가 / 감소) 연산자 a = a + 1 / a = a - 1 -> 숫자 뒤에 증가 및 감소 연산자가 오면 다른 실행을 하고, 연산을 실행 a(i++) -> a()를 실행시키고 i를 1 증가
a(i--) -> a()를 실행시키고 i를 1 감소

비교 연산자

참과 거짓을 확인하는 것

operator(연산) 이름 설명 예시
== 같음 두 수가 같은지 비교 10 == 6 + 4 ()
10 == "10" ()
!= 같지 않음 두 수가 다른지 비교 10 != 6 + 4 ()
10 != "10" ()
=== 같음(엄격하게 비교) 두 수가 완전히(값, 타입) 같은지 비교 10 === new Number(10) ()
10 === "10" ()
!== 같지 않음(엄격하게 비교) 두 수가 완전히(값, 타입) 다른지 비교 10 !== new Number(10) ()
10 !== "10" ()
< 미만 왼쪽의 값이 오른쪽 값보다 작은지 비교 5 < 3 ()
<= 이하 왼쪽의 값이 오른쪽 값보다 작거나 같은지 비교 3 <= 5 ()
> 초과 왼쪽의 값이 오른쪽 값보다 큰지 비교 5 > 3 ()
>= 이상 왼쪽의 값이 오른쪽 값보다 크거나 같은지 비교 3 >= 3 ()
