문자들의 집합 문자열 String

String 객체는 문자들의 집합인 문자열을 조작하는 함수를 제공

문자열 생성 방법

  • let x = "abc"; (기본데이터 타입)
  • let x = 'abc'; (Javascript는 char 타입이 없음)
  • let x = `abc`; (literal template 적용 문자)

문자(char) 접근하기

  • "string"[2] :
    (but "string"[2] = "a" (X))
  • console.log("String") -> charAt()없음
    console.log(new String("String")) -> charAt()있음

기본 데이터 타입인 문자열과 인스턴트 객체인 문자열

  • let stringVal = "Hello"; (기본 데이터 타입인 문자열)
  • let objVal = new String("Hello"); (new 연산자로 생성한 인스턴트 객체)
  • typeof stringVal :
  • typeof objVal :
  • stringVal == objVal : (value는 같음)
  • stringVal === objVal : (type은 다름)
  • 기본 데이터 타입인 문자열에는 함수가 없으나 사용할 수 있는 이유는 인스턴트 객체 생성과 비슷한 일이 발생하기 때문

Escape Sequence

특수 문자나 백슬래시(\), 줄 바꿈, 따옴표 같은 것을 출력할 수 있음

  • \"문자\":
  • \'문자\':
  • \\문자\\:
  • \n:
  • \t:
  • \b:

template literal ${표현식}

문자열을 ``(백틱)으로 선언하면 template literal 표현식(${ })을 사용한다는 의미 이때 Javascript 문법을 표현식에 사용가능
