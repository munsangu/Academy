Number 객체의 전역 변수와 함수

기본 데이터 타입인 수는 Number라는 타입의 객체를 가짐 이때 기본 데이터 타입을 보조하기 위해 Number가 내장 객체(New 연산자 없이 사용할 수 있는 객체)로 존재하며 다양한 전역 변수(어디든 사용할 수 있는)와 메소드를 제공하고 있음

내장 객체 Number의 전역 변수

변수명 설명
Number.MAX_VALUE 표현가능한 가장 큰 수(e+308)
Number.MIN_VALUE 표현가능한 (소수점이)가장 작은 수(e-324)
Number.NaN 숫자가 아닌 값을 검사(하지만 타입은 Number), 수가 아닌 것을 연산할 때
Number.MAX_SAFE_INTEGER 숫자가 표현 가능한 가장 큰 정수(변경되지 않는 수)
Number.MIN_SAFE_INTEGER 숫자가 표현 가능한 가장 큰 정수(변경되지 않는 수)
