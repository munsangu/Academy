for ~ in 반복문

for(variable in object) {실행문} : variable을 가진 array, object ...가 variable을 반환할 수 없을 때 까지 실행문을 반복

  • ["a", "b", "c "] ->
  • {name: "Tony Stark", age: 100, hero: "IRONMAN"} ->

for ~ of 반복문

for(variable of iterable) {실행문}: 반복 가능한 데이터 타입의 객체를 반복하고 각 개별 속성값에 대해 실행되는 문이 있는 사용자 정의 반복 후크를 호출하는 루프를 생성

(iterable).next( ) : next( )는 포인트가 객체 밖에 있다가 내용물이 있으면 true를 반환하고 내용물로 포인트를 옮김. 그리고 내용물도 반환, 만약 값이 더 이상 나오지 않으면 false를 반환
