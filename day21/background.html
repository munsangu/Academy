토끼

background-image는 해당 요소의 스타일 속성

background-clip : color를 나타내는 영역(default(border-box), padding-box, content-box)

content-box : 패딩을 제외한 영역에 color를 나타냄

padding-box : 패딩까지의 영역에 color를 나타냄

background-clip : color를 나타내는 영역(default(border-box), padding-box, content-box)

border-box(default) : 테두리 영역에 color를 나타냄

background-clip : color를 나타내는 영역(default(border-box), padding-box, content-box)

content-box : 패딩을 제외한 영역에 color를 나타냄

background-origin: border-box -> 테투리 영역까지 image로 나타냄

background-origin: content-box -> 내용 영역까지 image로 나타냄

background-origin: default(padding-box)
