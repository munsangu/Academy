Javascript 조건문

블럭 안의 실행을 조건에 따라 동적으로 선택되게 하는 조건문 if, switch, 삼항 연산자

if 조건문

if(조건문) {실행문} : 조건이 true이면, 블럭({ })을 실행

if(조건문) {실행문1} else {실행문2} : 조건문이 true이면 실행문1 을 출력, 조건이 false이면 실행문2 를 출력

if(조건문1){실행문1}else if(조건문2){실행문2}else{실행문3}: 조건문1 이 true이면 실행문2를, 조건문1이 false이고 조건문2가 true이면 실행문2, 조건문1과 조건문2 둘 다 false이면 실행문3

  • if(value < 50) { } :

    • if(value > 20 && value < 50) { } :

      • if(value < 20 || value> 50) { } :

        • if ((value >= 10 && value <= 20) || (value>= 30 && value <= 40)) { } :

Switch 조건문

작성법: switch (표현식 or 변수) {case1: 실행문1 break; case2: 실행문2 break; ... default: 실행문3 }

switch ~ case 는 엄격한 비교(===)를 사용하고 위에서부터 일치하는 항목을 찾으면 해당 실행을 하고 종료됨(엄격한 동등 비교 외 조건 사용 불가)

① if에서 else if문과 동일하게 작동하지만 조건문의 수행 속도가 빠름
② 조건을 파악하기 좋음

삼항 연산자

(condition) ? true : false => 조건문이 true이면 true 를 false이면 false를 출력

조건 파악이 편하고 코드가 간결해지기에 프로그래밍 속도를 올려줌
