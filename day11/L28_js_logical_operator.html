Javascript logical operator

&& (and), || (or), ! (not)

||(or) - 합집합을 의미, 하나의 true만 있으면 true 출력, 모두 false면 false 출력

&& (and) - 교집합을 의미, 하나의 false만 있으면 false 출력, 모두 true면 true 출력

논리 연산의 우선순위

&&와 ||는 우선 순위가 있음 ( ①&&<*> ② ||<+>)
