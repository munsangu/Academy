HTML 규칙

  • 태그를 HTML의 요소라고 하고, 시작과 끝이 있음 < >: 열고(시작) < / >: 닫음(끝)
  • br, a, img, input, meta 등의 태그는 내용이 없어서 닫는 태그가 없음
    하지만, 태그 끝에 / 를 사용해 닫는 강박성은 존재
  • 태그는 약속된 것만 사용, 소문자로 작성하는것을 권장
  • img, a, meta와 같이 독특한 속성을 가지는 태그가 존재
  • 루트 요소(HTML)를 가지는 구조적 형태를 tree구조라고 함
    HTML의 구조를 DOM(Document Object Model)tree라고 함
