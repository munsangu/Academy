HTML 기본 요소

기본 요소는 사용빈도가 가장 많은 태그로 p, br, h1 ~ h6, hr, 주석이 있음

  1. p 태그는 단락태그로 문서의 한 단락을 표현

    단락으로 선택된 영역은 위아래로 여백(margin)이 있는 블록이 형성됨

  2. br 태그는 single line break 줄바꿈을 의미 Enter를 이용한 줄바꿈은 개발의 편의를 위해 존재 이 동작은 브라우저가 인지하지 못함
    내용이 없기 때문에 단독으로 사용되고 닫힘 태그가 없음
  3. hn(n= 1 ~ 6)은 머릿말을 표현하는 명시적 태그, 글자 크기

    h1 머릿말

    h2 머릿말

    h3 머릿말

    h4 머릿말

    h5 머릿말
    h6 머릿말
    hn(n= 1 ~ 6) 태그는 단락 태그로 p태그와 동일하나 머릿말이라는 명시적 의미를 가짐
  4. hr 태그는 수평션(Horizontal) 태그로 두께가 1px의 수평선을 만듦.
    수평선의 실체는 테두리(border)와 위아래 여백(margin)을 가지는 블록
  5. 주석(< !-- -- >)은 구현되지 않는 코드, 잘못 사용시 오류가 발생할 수 있음
