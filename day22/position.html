position 요소, node 객체의 위치를 정의
(static, relative, absolute, fixed, sticky)

position: static (default)

모든 요소, node 객체는 위치를 static으로 정의함

  • 요소의 배치는 순차적(↓)이며 왼쪽에서부터 오른쪽으로 배치됨
  • 자식 요소는 부모 요소의 위치를 기준으로 배치됨
  • top, right, bottom, left 사용 불가

position: relative (상대적 위치)

relative

-> 자식 박스를 상대적 위치로 top 25% / left 25% 이동

  • 상대적 위치 relative는 정적 위치 static을 기준으로 node를 이동 시킴
  • 상대적 위치로 이동한 node 객체는 다른 node에 영향을 주지 않음

position: absolute (절대적 위치)

absolute (부모 : static)

position: absolute (절대적 위치)

absolute (부모 : relative)

  • absolute는 조상 요소 중 relative, absolute, fixed가 정의 되어 있으면 그 조상을 기준으로 위치를 이동시킴
  • 만약 부모, 조상 중에 relative, absolute, fixed가 없다면 body를 기준으로 위치가 조정됨
  • 부유객체로 다른 요소를 밀거나 당기지않고 덮어쓰는 형태로 존재함
  • absolute로 정의된 node는 너비가 content에 맞게 지정되므로 너비를 지정하는 것이 좋음

position: fixed

fixed
  • 부모 요소와 아무런 상관없이 viewport(브라우저화면)를 기준으로 node 객체를 이동시킴
  • 너비와 높이가 content에 맞게 변경됨
  • 부유객체로 다른 node 객체를 덮음
