html의 화면을 구성하는 block 요소들

상용 가능한 전체 넓이(width)와 컨텐츠 만큼의 높이(height)를 갖는 사각의 박스로 여러 요소를 구조적으로 구성 가능

스타일로 본 block의 box model

Box Model
width: 150px
height: 150px
padding: 20px
border: 10px
margin: 40px auto
블럭을 정렬

box model 정리

  • box contents : 점선
  • padding : 점선과 파란선 사이
  • border : 파란선
  • margin : 회색 여백
  • block default 정렬(왼쪽 정렬) : margin-right: auto
  • block 오른쪽 정렬 : margin-left: auto
  • block 중앙 정렬 : margin: center

display: block 속성을 갖는 요소들(tag)

num 태그(tagname) 설명(Description) 기본스타일(Default style)
01 < div > 모든 요소의 부모인 컨테이너로 요소를 구조적으로 만들때 사용 display: block;
02 < header > 제목, 저작권, 로고, 작성자 등을 표시하고 article의 헤드라인으로 사용 display: block;
03 < main > 문서에 하나 있기를 기대하며 페이지의 컨텐츠를 모두 포함하는 영역 display: block;
04 < footer > 저작권, 연락처, 소셜미디어, 약관, 관련된 문서 display: block;
05 < section > 주제를 갖는 문서의 한 구문으로 여러 article을 포함 display: block;
06 < article > 작은 단위의 내용을 포함하는 구문 display: block;
07 < aside > 주변 콘텐츠와 간접적으로 관련되는 내용으로 화면의 사이드로 배치하는 편 display: block;
08 < nav > 탐색 링크의 집합인 navigation, 대체로 메뉴 리스트를 구성 display: block;
09 < address > 이메일 주소, URL, 실제 주소, 전화번호, 소셜 미디어 같은 정보를 정의 font-style:italic
10 < form > 유저가 입력할 폼을 생성하는데 사용 margin-top: 0em
11 < figure > 그림, 도표, 사진, 코드 목록을 구성 margin: 1em 40px
12 < blockquote > 다른 소스에서 인용된 부분을 지정 display: block;
13 < ul,ol,dl > 목록 구성하며 스타일 각각 다름 list-style-type:(disc,decimal) margin: 1em 0px; padding-left: 40px;
14 < p > 문서의 한 문단을 정의 margin: 1em 0
15 < h1 ~ h6 > 문서의 제목이며 한 문단을 정의 h1은 검색에 활용되므로 꼭 문서에 1개를 생성 margin:(0.67~2.3)em 0; font-size:(2~0.67)em
16 < pre > 태그 내의 텍스트는 html 소스 코드에 작성된 그대로 표시됨 margin: 1em 0
