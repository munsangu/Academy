html은 색을 광원의 3색을 조합하는 원리로 표현

RGB -> Red Green Blue

RGB 색 모형은 빛의 3원색을 이용하여 색을 표현하는 방식 빨강(Red), 초록(Green), 파랑(Blue) 세 종류의 광원을 이용하여 색을 혼합, 색을 섞을수록 밝아지기 때문에 '가산혼합'이라고 함

빨간색, 초록색, 파란색은 0 ~ 255 사이의 값으로 색상의 강도를 나타내며 256 * 256 * 256 = 16,777,216개의 색을 표현할 수 있음

16진수 Hexadecimal color #RRGGBB

RGB의 0 ~ 255 를 16진수 (0 ~ f) * (0 ~ f) = 256으로 표현

RGB와 완전히 동일하며 브라우저는 HEX를 RGB로 변형해 저장함.
