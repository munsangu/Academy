transition으로 작은 animation 만들기

transition은 css property 값이 변할때 하당 변화를 일정 시간동안 일어나도록 해서 animation을 만듦

Change backgroundColor
  • transition-property: animation이 가능한 css 속성 (default = all)
  • transition-duration: transition이 일어나는 시간 (default = 0)
  • transition-delay: transition이 시작하는 시간 (default = 0)
  • transition-timing-function: transition 효과를 위한 속도 조절 함수(default = ease)
  • ┗> cubic-bezier(), ease, linear, ease-in, ease-out, ease-in-out
