Javascript Prototype

  • 함수형 언어인 Javascript는 prototype 언어라고도 함
  • 함수를 선언하면 prototype이 생성되고 (설계도 역할인 prototype은) 서로 다른 prototype을 연결해서 상속을 대신함
  • prototype에 대한 이해의 필요성
    1. 객체 지향 언어의 상속과 개념이 다르기 때문에 Javascript 고급 수준에 오면 개념이 헷갈릴 수 있음
    2. 프로그래밍을 경험하지 못하는 사람들이 프로그래밍을 하기 위해 Javascript로 새로운 언어를 만들고 있기 때문에 prototype을 이해하면 Javascript로 만든 모든 언어를 이해할 수 있음
    3. prototype이 2차원적 개념이면, 객체 지향적 상속은 3차원적 개념
