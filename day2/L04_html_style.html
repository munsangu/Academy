HTML Style CSS(Cascading Style Sheets)

Style CSS는 레이아웃, 색상, 텍스트, 크기, 이미지, 배경, RWD(Responsive Web Design) 등 html의 요소가 화면에 표시되는 모든 방식을 제어

CSS 구문(Syntax)

  • Selector { property: "value"(X);}
  • Selector { property: value(O);}
  • 선택자 {요소: 값; 요소: 값; 요소: 값; ...... }
  • tagname style="property:value; property:value;"
  • ex) h1 {color: red; background-color: gray; ...}
  •  h1 {
      color: red;
      background-color: gray;
      color: black; // (나중에 선언한 것이 우선 적용)
    }

HTML < = CSS 적용 방법 3가지

  1. 외부 CSS: head 태그에 link 태그 사용(link rel="stylesheet" href="경로")
  2. 내부 CSS: head 태그에 style 태그 사용(style) selector {property:value; ...}(/style)
  3. 인라인 CSS: (tagname style="property:value;")(/tagname)
  4. 요소에 동일한 스타일이 다르게 제공 되었을 때 우선 순위

    5. !important > 인라인 > 내부 > 외부
