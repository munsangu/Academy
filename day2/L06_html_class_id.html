HTML class & id

모든 html 요소는 class와 id 속성을 가짐, 두 속성은 선택자로 이용됨

복수의 선택자 class

모든 html 요소에 하나 이상의 이름을 가지는 것을 class, 중복 가능

  1. class="이름1"
  2. class="이름1 이름2 이름3 ..."
  3. .이름{ property: value; ... }
  4. document.getElementByClassName("className")[n]
  5. document.querySelector(".class")[n]
  6. class는 복수의 자료(Document Object) 처리할 때 사용

유일한 선택자 id

  1. id = "idName" (O)
  2. id = "id_name" (△)
  3. id = "name" (△)
  4. id = "011id" (△)
  5. id를 만들 때는 단어와 단어의 조합으로, 시작은 소문자로 조합되는 단어는 대문자로 작성
    (△ -> 작동은 되나 권장하지 않는 작성법)
    권장하는 방식이 document에서 유일할 가능성과 가독성이 높음
  6. #이름{property: value; ...}
  7. document.getElementById("idName")
  8. document.querySelector("#id")
