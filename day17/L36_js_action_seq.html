HTML 문서에 선언된 스크립트의 동작 순서

  1. 브라우저는 HTML 문서를 CSS와 병합해 DOMTree 구조를 생성
  2. 브라우저가 병합된 구조를 보고 화면에 그래픽으로 객체를 표현(순차적 언어)
  3. 순서대로 객체를 출력하던 중 Javascript 구문을 발견하면 순서대로 그것을 실행함
  4. 이때 Javascript가 node 객체를 검색하던 중이면 브라우저가 해당 node 객체를 출력해야 찾을 수 있음

순차적 언어인 HTML과 Javascript가 문제를 일으키기 않게 하는 방법

  1. Javascript를 모든 node가 생성된 가장 아래에 작성
  2. onload : 해당하는 node가 생성되었는지 확인 / window.onload = function() { }
  3. window 객체가 완전히 생성된 시점 즉, node가 모두 load된 시점에 Javascript를 작성하면 됨
  4. <script defer src=" "></script>: HTML문서를 다 다운받고 load한 시점에 참조된 문서를 다운
